# Où est le BDE MIST dans la vie ?

### Utilité

Petit site cool qui permet de voir où est-ce que vos potes seront dans les mois a venir.
Pour l'instant il permet aux gens de remplir pour 
 - Cet été : stage 
 - Le semestre prochain (premier semestre 2023-2024) : cesure

### Données

Pour fonctionner faut les données (ville + pays) des personnes, à remplir dans `infos.txt` en suivant les consignes.
